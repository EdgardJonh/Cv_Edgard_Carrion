PERFIL

Soy un desarrollador web proactivo con una sólida formación técnica y una fuerte pasión por la tecnología. Cuento con experiencia en desarrollo Fullstack y Frontend, respaldada por tres bootcamps especializados: Desarrollador Fullstack JavaScript, Desarrollador Fullstack Python y Desarrollador Frontend con Vue.js. He trabajado en proyectos significativos como un marketplace de comida y un sistema de gestión de inventarios, utilizando React.js y otras tecnologías avanzadas. Mis habilidades incluyen el desarrollo bajo el patrón de diseño Modelo-Vista-Controlador (MVC), creación de sistemas de gestión de información (CRUD), mapeo relacional de objetos (ORM) y bases de datos como PostgreSQL. Además, tengo experiencia con Node.js, JWT, middleware, consumo de APIs REST y despliegue de sistemas en producción en la nube con Render. Domino herramientas y entornos como GitHub para control de versiones y sistemas operativos Windows y Linux. En el Frontend, destaco en la creación de interfaces responsive usando Bootstrap, Flexbox y CSS Grid, ofreciendo soluciones intuitivas y visualmente atractivas. Mi enfoque se complementa con conocimientos en lenguajes como JavaScript, PHP y Visual Basic .NET, además de una mentalidad orientada al aprendizaje continuo. Mi lema, "La tecnología son solo herramientas para generar soluciones", guía mi enfoque práctico y centrado en resultados.
